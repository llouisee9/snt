Chapitre : Python

1. Qu’est-ce que Python ?

Python est un langage de programmation inventé en 1991 par Guido Van Rossum. C'est un langage interprété, multiplateforme et libre. Python existe en deux versions majeures : Python 2 et Python 3 (non compatibles entre elles).

2. L’interpréteur Python

L'interpréteur exécute les instructions au fur et à mesure. Le symbole >>> indique que Python attend une commande.

3. Programmer avec un éditeur

Pour des programmes plus longs, on crée des fichiers .py appelés scripts. Exemple :

# helloWorld.py
print("Hello World !")

4. Les variables

Une variable est un espace mémoire qui stocke une valeur. Les noms ne doivent pas commencer par un chiffre, ne pas contenir d’espace, ni utiliser de mots réservés (ex : print). Python est sensible à la casse.

5. Types de données

  • Entier : a = 20
  • Réel : pi = 3.14159
  • Chaîne de caractères : msg = "Bonjour"
  • Liste : tab = [9, 7, 6]

6. Affichage en console

La fonction print() permet d’afficher des valeurs.

print("Bonjour")
print("Valeur de a :", a)

7. Saisie clavier

message = input("Entrer un nom : ")
a = int(input("Entrer un entier : "))
b = float(input("Entrer un réel : "))

8. Instructions conditionnelles

Les structures conditionnelles s’écrivent ainsi :

if condition:
    instruction
elif autre_condition:
    instruction
else:
    instruction

9. Boucles

  • Pour : utilisée quand on connaît le nombre de répétitions
    • for i in range(5):
    print(i)
  • Tant que : utilisée quand on ne connaît pas le nombre de répétitions
    • while condition:
    instruction
