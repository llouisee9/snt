Chapitre : Données structurées et leur traitement

Point de cours 1

  • Données : Une donnée est une valeur décrivant un objet, une personne, un événement digne d'intérêt.
  • Descripteurs : Ce sont des catégories de données utilisées pour décrire un objet (ex : chanson).
  • Données structurées : Une collection d'objets ayant les mêmes descripteurs organisés sous forme de table.
  • Information : Les données structurées peuvent être interprétées pour extraire de l'information utile.

Point de cours 2

  • Format de fichier : Les données sont conservées sous différents formats pour assurer leur durabilité.
  • Format CSV : Fichier texte avec des données tabulaires, séparées par virgule, point-virgule ou tabulation.
  • Format ouvert : Un format public, sans restrictions légales, librement utilisable.
  • Interopérabilité : Les formats ouverts assurent une compatibilité entre logiciels différents.
  • Métadonnées : Informations descriptives sur les données d’un fichier (titre, auteur, date, etc.).

Point de cours 3

  • Bandes magnétiques (1950) : Stockage séquentiel sur bande magnétique.
  • Disquettes (années 1970) : Supports magnétiques portables avec capacité limitée.
  • Disques durs (années 1980) : Plateaux magnétiques à grande capacité et accès rapide.
  • SSD (années 2000) : Mémoire flash rapide et économe en énergie.
  • Stockage en ligne (années 2010) : Données hébergées sur serveurs distants accessibles via internet.

Point de cours 4

  • Recherche dans une table : Opérations de tri, filtrage, calcul et mise en forme sur les données tabulaires.
  • Recherche dans une base de données : Utilisation de SGBD pour requêtes et génération d'information.
  • Exemple musical : Base de données séparant morceaux et interprètes pour éviter les doublons.

Point de cours 5

  • Big Data : Volume croissant de données, exploité par l'IA et le calcul intensif.
  • Open Data : Données ouvertes mises à disposition par les services publics (ex : Covidtracker, Suptracker).
  • RGPD : Règlement européen sur la protection des données personnelles.
  • Impact environnemental : Les data centers consomment beaucoup d’énergie et ont un impact écologique important.
