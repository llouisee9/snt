Chapitre : Internet

Sciences Numériques et Technologique - Lexique

  1. Processus informatique
    Un processus (en anglais, process), en informatique, est un programme en cours d'exécution par un ordinateur.
  2. Protocole
    Un protocole informatique définit les règles et les procédures permettant à deux processus informatiques d’échanger des données, notamment à travers un réseau.
  3. TCP/IP
    TCP/IP est un protocole, c'est à dire des règles de communication.
    • IP
      IP signifie Internet Protocol : littéralement "le protocole d'Internet". C'est le principal protocole utilisé sur Internet. Internet signifie Inter-networks, c'est à dire "entre réseaux". Internet est l'interconnexion des réseaux de la planète. Le protocole IP permet aux ordinateurs reliés à ces réseaux de dialoguer entre eux.
    • TCP
      TCP est capable :
      • de faire tout ce que UDP sait faire (ports)
      • de vérifier que le destinataire est prêt à recevoir les données
      • de découper les gros paquets de données en paquets plus petits pour que IP les accepte
      • de numéroter les paquets, vérifier qu'ils sont tous bien arrivés, redemander les paquets manquants et les réassembler
      • d'envoyer des accusés de réception
  4. Paquet
    Les ordinateurs s'échangent des données de taille variable (fichiers, pages web, etc.) mais le réseau gère uniquement des paquets de taille fixe (MTU). Ces paquets sont appelés datagrammes ou segments.
  5. Adresse symbolique
    Un nom de site, plus facile à mémoriser que l'adresse IP, comme www.yahoo.fr. Il est converti par le DNS en adresse IP (ex. : 87.248.98.7).
  6. DNS
    DNS (Domain Name System) est un serveur qui remplace automatiquement l'adresse symbolique par l'adresse IP correspondante.
  7. Routeur
    Le routeur détermine le chemin que les paquets empruntent pour arriver à leur destination. Il est connecté à plusieurs réseaux pour faire transiter les données.
  8. Client/serveur
    Le modèle client-serveur décrit une communication réseau où un client envoie des requêtes à un ou plusieurs serveurs, qui y répondent.
  9. HTTP/HTTPS
    HTTP est un protocole de communication client-serveur pour le web.
    HTTPS est la version sécurisée de HTTP (HyperText Transfer Protocol Secure).
  10. NTP
    NTP (Network Time Protocol) permet de synchroniser l'heure des ordinateurs via un réseau.
  11. Réseau pair à pair
    Le modèle pair à pair (peer-to-peer, P2P) est similaire au client-serveur, sauf que chaque client peut aussi agir comme serveur.
