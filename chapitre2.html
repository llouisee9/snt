Chapitre : Le Web

  1. Définition
    Le web est l’abréviation de World Wide Web. C’est un système hypertexte qui fonctionne sur le réseau internet. Les pages disponibles sur le web sont liées entre elles par des liens hypertextes, comme une toile d’araignée.
    Remarque : Ne pas confondre le web (service) avec internet (réseau).
  2. Structure client-serveur
    Un serveur offre des services, un client les demande via un logiciel (navigateur). Le client émet une requête vers le serveur via une adresse et un port. Le serveur répond au client. L’interface graphique d’accès est le navigateur web.
  3. L’URL
    Une URL (Uniform Resource Locator) est une adresse unique liée à une ressource (page HTML, image, vidéo, etc.). Une URL contient :
    • Le protocole (HTTP, HTTPS, FTP...)
    • Le nom de domaine (adresse web)
    • Le chemin d’accès au fichier
    Exemple : http://www.jcbmathsandco.fr/snt-sciences-numeriques-et-technologie-le-web/
  4. Le protocole HTTP
    HTTP est le protocole de base du Web pour afficher les pages. HTTPS (version sécurisée) utilise TLS pour protéger les échanges. Un site HTTP est vulnérable aux piratages, contrairement à HTTPS (cadenas dans le navigateur).
  5. Les moteurs de recherche
    Les moteurs comme Google ou Bing permettent de retrouver des pages web. Les sites sont indexés par des robots d’indexation, qui lisent le contenu et parcourent les liens. Le SEO (Search Engine Optimization) permet d’améliorer la visibilité des sites.
  6. Impact sur les pratiques humaines
    Le Web est devenu essentiel :
    • Première source d’informations
    • Outils de communication (réseaux sociaux)
    • Explosion du commerce en ligne
